Set the current table. 

All of the functions Insert(), Update(), Delete() and Select() execute for the currently selected table. If no table is selected the functions give an error.

To select a table use the \hyperlink{classTiny_1_1TinyDB_ac65bbcea4f2ffb104987aa703c49d1bc}{Open()} method. Only one table can be selected(opened) at any given time. To close a table use the \hyperlink{classTiny_1_1TinyDB_a677ece6b542a1ffb980e900deb6723a1}{Close()} method.
