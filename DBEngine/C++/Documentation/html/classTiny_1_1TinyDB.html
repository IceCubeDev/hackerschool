int Tiny::TinyDB::Open ( const std::string &  table)

Set the current table.

All of the functions Insert(), Update(), Delete() and Select() execute for the currently selected table. If no table is selected the functions give an error.

To select a table use the Open() method. Only one table can be selected(opened) at any given time. To close a table use the Close() method.

Parameters
tableThe name of the table to open.
Returns
Returns true on success and a negative number on failure.
